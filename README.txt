== Description ==

This plugin allow you to extract and add files in the media library from a zip file.

You juste need to upload or select an already uploaded zip file and the plugin will extract all the file and add them in the media library.

== Installation ==

1. Upload `upunzipper.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
